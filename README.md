# SpringBoot-Web-02
这是我根据SpringBoot-Web-01仓库里的小项目写的，主要展示一个简单的前端页面。

## 运行方式
1. 克隆仓库：git clone https://github.com/tianbohao/SpringBoot-Web-02.git
2. 打开项目（如 IntelliJ IDEA）
3. 运行主类即可启动项目
4. 浏览器输入localhost:8080/user.html，回车

## 技术栈
- SpringBoot 2.x
- Java 8+
